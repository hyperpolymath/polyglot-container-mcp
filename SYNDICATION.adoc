=== Full Description
----
polyglot-container-mcp is a unified Model Context Protocol (MCP) server that provides AI assistants with seamless access to multiple container runtimes through a single, consistent interface.

**Supported Runtimes:**
- **nerdctl**: containerd CLI (FOSS preferred)
- **podman**: Daemonless container engine (FOSS preferred)
- **docker**: Docker CLI (fallback for compatibility)

**Features:**
- 95+ container management tools exposed via MCP
- FOSS-first: nerdctl and podman preferred over Docker
- Nested container support (DinD, podman-in-podman)
- Container, image, network, volume, compose operations
- Multiple FOSS base images (Wolfi, Alpine, Fedora)
- Type-safe ReScript core

**Installation:**
```bash
# Direct
deno run --allow-run --allow-read --allow-env \
  https://raw.githubusercontent.com/hyperpolymath/polyglot-container-mcp/main/index.js

# Container
podman run --rm -it ghcr.io/hyperpolymath/polyglot-container-mcp:wolfi
```
